Launch The EC2 Instance and Connect the private instance. Before that we need to create a VPC,Subnets and Route Table and make it as a public and private.


Step 1:Go to VPC service on aws console and you can find the VPC dashboard. In VPC dashboard Click Create VPC.


Step 2:Give the tag name and CIDR block as per the below. Click Create.


Step 3:You can find that VPC has been created.


Step 4:Then we need to create a subnets for public and private. Go to Subnets on same VPC dashboard and Click Create subnet.


Step 5:Give the Tag name as public and choose your created VPC, select the AZ and Need to specify the subnets IP block in CIDR block. Click Create.


Step 6:Need to follow the same step to cretae the private subnet. Go to Subnets on same VPC dashboard and Click Create subnet.

Step 7:Give the Tag name as private and choose your created VPC, select the AZ and Need to specify the subnets IP block in CIDR block. Click Create.
